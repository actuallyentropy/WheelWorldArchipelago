Archipelago client for Wheel World. This should be used in conjunction with https://github.com/actuallyentropy/WheelWorldAPWorld.

Currently box pickups, race rewards, and shop items are supported.

This is in early development. https://github.com/alwaysintreble/ArchipelagoBepInExPluginTemplate was used to start the project, and implementation was performed with heavy Claude use. Expect bugs.
